# Русская Ясна — лендинг

Открытое сообщество исследователей русского языка, истории и культуры. Восемь направлений, общие праздники, совместный поиск.

## Структура проекта

```
.
├── index.html                ← главная (новая версия)
├── index v1.html             ← старая версия (до аудита, для отката)
├── pages/
│   └── neglinka.html         ← внутренняя страница направления
└── src/
    ├── styles.css            ← базовая дизайн-система (палитры, токены)
    ├── refresh.css           ← новая типографика + переработанные блоки
    ├── interactive.jsx       ← React-компоненты: Cygnus, Quiz, Cross
    ├── image-slot.js         ← drop-зона для пользовательских изображений
    ├── tweaks-panel.jsx      ← плавающая панель настройки
    └── tweaks-app.jsx        ← конфиг палитр/шрифтов/плотности
```

## Дизайн-система

### Палитра — Иней (по умолчанию)

| Токен          | HEX        | Назначение                       |
|----------------|------------|----------------------------------|
| `--bg`         | `#f5f3ee`  | основной фон                     |
| `--bg-deep`    | `#ebe7df`  | фон контрастных секций           |
| `--paper`      | `#fbfaf6`  | поверхность карточек             |
| `--ink`        | `#1a1714`  | основной текст                   |
| `--ink-2`      | `#4a443d`  | вспомогательный текст            |
| `--ink-3`      | `#8a847a`  | мета / mono / placeholder        |
| `--accent`     | `#a9743b`  | охра — акценты, italic, активное |
| `--accent-2`   | `#2f4170`  | глубокий синий                   |

Альтернативные палитры доступны через `[data-palette]`: `iney` · `zemlya` · `lazur` · `ugol` (тёмная).

### Типографика

- **Spectral** (Google Fonts) — заголовки, акценты, italic для смысловых выделений
- **IBM Plex Sans** — основной текст, кнопки, навигация
- **IBM Plex Mono** — meta-надписи, eyebrow, числа, технические подписи

Размеры через `clamp()` — адаптивно от 320 до 1920px.

### Сетка и ритм

- Контейнер: 1240px
- Гаттер: `clamp(20px, 4vw, 56px)`
- Радиусы: `--radius: 14px`, `--radius-lg: 22px`
- Вертикальный ритм секций: 88px (`tight`-секции: 48px)

## Структура лендинга

1. **Hero** — центрированный, fits-one-screen, манифест-карточка справа
2. **День Ясны** — главное событие года (открытое, бесплатное)
3. **Восемь направлений** — типографические карточки с кириллическими литерами
4. **Метод** — как живёт сообщество
5. **Четыре опоры. Один круг.** — пример мышления (Russian Cross)
6. **FAQ** — частые вопросы
7. **CTA** — заявка + Telegram

## Подключение шрифтов

```html
<link href="https://fonts.googleapis.com/css2?family=Spectral:ital,wght@0,400;0,500;0,600;0,700;1,500;1,700&family=IBM+Plex+Sans:wght@400;500;600;700&family=IBM+Plex+Mono:wght@400;500;600&display=swap" rel="stylesheet"/>
```

## Запуск локально

Просто откройте `index.html` в браузере. Шрифты тянутся с Google Fonts, React — с unpkg.

Для нормальной работы `image-slot` (drag-and-drop фотографий с сохранением) сайт должен открываться через локальный HTTP-сервер:

```bash
python3 -m http.server 8000
# или
npx serve .
```

## Tweaks-панель

В правом нижнем углу — плавающая панель с переключателем палитры, шрифтовых пар и плотности. Для prod рекомендуется закомментировать подключение `tweaks-app.jsx` в `index.html`.
